# Projeto_Diabrera
Projeto integrador 3º semestre, curso ADS

# Descrição :

O sistema é um site web para desktop e mobile para expor descrições breves sobre o tipo de serviço (tarot), preços, agenda, e deve ter um espaço reservado para redirecionamento para o whatsapp da provedora de serviço. O site deve seguir a paleta de cores vermelha e preto com sombreados translucidos e seguir a ideia de scroll “infinito” para da a impressão de livro. O sistema deve ser capaz de expor ao cliente interessado o tipo de serviço e redireciona-lo para o whatsapp da cartomante. Além disso, o sistema deve ser capaz de criar uma atmosfera semelhante ao instagram da provedora para criar um ar de pertencimento. O sistema deve ser capaz de expor serviços disponiveis, agenda, whatsapp, informações sobre o que a cartomante faz e um pouco da sua historia e experiência com tarot. Para o usuário administrador deverá ser exibido telas com dias preenchidos da agenda, quantidade de clientes e especulação de lucros em gráfico pizza.
